1%LOAD Load data from MAT-file into workspace.
 2%   S = LOAD(FILENAME) loads the variables from a MAT-file into a structure
 3%   array, or data from an ASCII file into a double-precision array.
 4%
 5%   S = LOAD(FILENAME, VARIABLES) loads only the specified variables from a
 6%   MAT-file.  VARIABLES use one of the following forms:
 7%
 8%       VAR1, VAR2, ...          Load the listed variables.  Use the '*'
 9%                                wildcard to match patterns.  For
10%                                example, load('A*') loads all variables
11%                                that start with A.
12%       '-regexp', EXPRESSIONS   Load only the variables that match the
13%                                specified regular expressions.  For more
14%                                information on regular expressions, type
15%                                "doc regexp" at the command prompt.
16%
17%   S = LOAD(FILENAME, '-mat', VARIABLES) forces LOAD to treat the file as
18%   a MAT-file, regardless of the extension.  Specifying VARIABLES is
19%   optional.
20%
21%   S = LOAD(FILENAME, '-ascii') forces LOAD to treat the file as an ASCII
22%   file, regardless of the extension.
23%
24%   LOAD(...) loads without combining MAT-file variables into a structure
25%   array.
26%
27%   LOAD ... is the command form of the syntax, for convenient loading from
28%   the command line. With command syntax, you do not need to enclose input
29%   strings in single quotation marks. Separate inputs with spaces instead 
30%   of commas. Do not use command syntax if FILENAME is a variable.
31%   
32%   Notes:
33%
34%   If you do not specify FILENAME, the LOAD function searches for a file
35%   named matlab.mat.
36%
37%   ASCII files must contain a rectangular table of numbers, with an equal
38%   number of elements in each row.  The file delimiter (character between
39%   each element in a row) can be a blank, comma, semicolon, or tab.  The
40%   file can contain MATLAB comments.
41%
42%   Examples:
43%
44%       gongStruct = load('gong.mat')      % All variables
45%       load('handel.mat', 'y')            % Only variable y
46%       load('accidents.mat', 'hwy*')      % Variables starting with "hwy"
47%       load('topo.mat', '-regexp', '\d')  % Variables containing digits
48%
49%       % Using command form
50%       load gong.mat
51%       load topo.mat -regexp \d
52%       load 'hypothetical file.mat'       % Filename with spaces
53%
54%   See also SAVE, MATFILE, WHOS, CLEAR, REGEXP, IMPORTDATA, UIIMPORT.
